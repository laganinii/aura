## Aura iOS Prototype

Drop these files into a new SwiftUI Xcode project targeting iOS 17+.

### Required Targets
1. Main app (`Aura`)
2. Widget Extension
3. Live Activity (recommended)

### Capabilities
- HealthKit
- Background Modes (Background fetch)
- ActivityKit

The prototype demonstrates real background behavior using BGAppRefreshTask, WidgetKit timelines, and simulated Health signals. The generative art in EcosystemView matches the web demo's aesthetic.
